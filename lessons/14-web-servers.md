# Lesson 14: Web Servers (Nginx)

## 🎯 Objective
Install, configure, and verify a professional-grade web server using Nginx on Ubuntu.

## 🧠 Core Concepts
* **Web Server:** A service that listens for HTTP requests and serves web content.
* **Systemd (`systemctl`):** The tool used to manage services (start, stop, status).
* **Port 80:** The default standard port for unencrypted web traffic.

## 🛠️ Commands Learned
* `sudo apt install nginx`: Installs the server.
* `systemctl status nginx`: Checks if the server is running.
* `curl localhost`: Tests the web server from inside the terminal.

## 📝 Laboratory Work
Successfully installed Nginx and modified the default `index.html` located in `/var/www/html/`. Verified the server is active and serving content locally.

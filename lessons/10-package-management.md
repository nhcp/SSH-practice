# Lesson 10: Package Management

## 🎯 Objective
Learn how to safely install, update, and remove software packages using the Advanced Package Tool (APT).

## 🛠️ Essential Commands
* `sudo apt update`: Refreshes the local list of available software from the internet.
* `sudo apt upgrade`: Installs the newest versions of all software currently on the machine.
* `sudo apt install <package>`: Downloads and installs a new program.
* `sudo apt remove <package>`: Uninstalls a program.

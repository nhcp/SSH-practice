# Lesson 1: Linux Navigation & Safe Zones

## Goal
Learn how to create a safe practice folder and navigate your terminal without getting lost.

## Exercise 1: Create a Safe Zone
We need a sandbox folder so we don't accidentally break anything.
1. Make a new folder named 'linux-practice':
   `cd`
2. Move inside that folder:
   `cd linux-practice`

## Exercise 2: The "Where am I?" Command
Whenever you feel lost in the terminal, use this command to print your current location.
1. Run the Print Working Directory command:
   `pwd`
   (You should see your location ends in /linux-practice)

## Exercise 3: The "What is here?" Command
Let's look around.
1. List all files and folders in your current location:
   `ls`
   (It will be blank, because the folder is empty!)

## Exercise 4: Go Back Home
Let's leave the practice folder.
1. Change directory to go back up one level:
   `cd ..`
2. Run `pwd` again to prove you moved back!

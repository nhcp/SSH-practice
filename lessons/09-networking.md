# Lesson 09: Networking Basics

## 🎯 Objective
Understand how to verify network connectivity and identify system IP addresses and open ports.

## 🛠️ Essential Commands
* `ip a`: View network interfaces and IP addresses.
* `ping`: Test connectivity to a remote host.
* `ss -tulpn`: View active network connections and listening ports.

# Lesson 13: Loops (Automation Power)

## 🎯 Objective
Learn how to use `for` loops to automate repetitive tasks across multiple items efficiently.

## 🧠 Core Concepts
* **For Loops:** Iterates through a list of items and executes a block of code for each.
* **Exit Codes ($?):** In Linux, a successful command returns `0`. We can use this inside loops to check the status of tasks.
* **The `do/done` block:** Just like `if/fi`, loops are wrapped in `do` and `done`.

## 🛠️ Laboratory Work
Created `multiping.sh`. This script loops through a list of web domains and uses `ping` to verify connectivity for each, providing a clean status report.
